# College Enrollment System

## Overview

The College Enrollment System is a Java console application developed to simulate the process of managing student course enrollment. The project demonstrates the practical implementation of Data Structures such as Binary Search Tree (BST), Queue, and Arrays.

---

## Features

- Add Courses
- Add Students
- Display Courses
- Search Course by ID
- Enroll Students
- Waitlist Management
- Display Student Details
- Console-Based User Interface

---

## Data Structures Used

### Binary Search Tree (BST)
- Stores course records.
- Efficient searching and insertion.
- Displays courses in sorted order.

### Queue
- Manages students waiting for available seats.
- Implements FIFO (First In, First Out).

### Arrays
- Stores students.
- Stores enrolled courses for each student.
